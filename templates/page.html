About the F# language

This site has been built to let you easily explore the F# language.

For more information about F#, check out the F# Software Foundation web site. The Learn F# page lists additional resources such as books. You can also join as a member and help us support F#!

About F# Katas Playground

The F# Katas Playground has been built by F# experts from fsharpWorks and the content has been contributed by the great F# community.

Some Katas are adapted from existing content and have an explicit license. All other content is licensed under Creative Commons Attribution license. The source code for the playground is on GitHub and is licensed under Apache 2.0.
